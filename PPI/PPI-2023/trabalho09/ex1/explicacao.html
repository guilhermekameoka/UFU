Ex1-hello-ajax

O exercício está fazendo uma requisição via XHR, no qual ele está requisitanto um conteúdo adicional e está devolvendo esse conteúdo adicional dentro da divDemo através do método xhr.resposeText

Essa requisição é feita quando o usuário clica no botão

Caso ocorrer algum erro de rede, será mostrado a mensagem de erro

Caso contrário, o conteudo adicional será adicionado

Ex2-busca-cidade

O exercício está fazendo uma requisição via XHR para obter informações do campo cidade do formulário quando usuário digita um determinado CEP

Caso esse CEP já esteja cadastrado, a requisição irá devolver a cidade correspondente

Caso contrário, irá devolver o campo cidade em branco

Ex3-recebendo-json1

O exercício está fazendo uma requisição via XHR para buscar o endereço - rua, bairo, cidade - de um determinado CEP

Só que diferentemente do exercício anterior, a string JSON retornada será automaticamente convertida em um objeto JS

Ex4-recebendo-json2

O exercício é similar ao anterior, só que nesse caso a conversão da string JSON é feita de maneira manual

Ex5-enviando-json

O exercicio converte o objeto JS para JSON

Depois altera o cabeçalho HTTP com o valor apropriado para eviar um JSON com os valores do CEP, email e apikey

Ex6-login

O exercício checa os dados se os dados de login são válidos e também garante que o formulário senja enviado pelo método sendForm
