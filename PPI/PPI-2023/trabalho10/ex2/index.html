Exemplo ex2-encadeamento-fetch

Explicação

A função buscaClimaLocal é chamada quando a janela é carregada (window.onload)

Em seguida, uma requisição assíncrona é iniciada usando o método fetch, que faz uma chamada para a URL da API de Geolocalização para obter informações sobre a localização do usuário

Após receber a resposta da primeira requisição, quatro métodos .then são encadeados para processar os dados

O primeiro método converte a resposta para o formato JSON. Em seguida, o segundo método utiliza as informações de latitude e longitude obtidas para fazer outra requisição assíncrona, desta vez para a API Open-Meteo, que fornece dados meteorológicos (temperatura e velocidade do vento)

Após receber a resposta da segunda requisição, o terceiro método .then converte os dados para o formato JSON

Em seguida, o quarto método .then extrai a temperatura e a velocidade do vento dos dados recebidos e atualiza o conteúdo dos elementos HTML com os IDs "temp" e "wind"

No caso de ocorrer algum erro em qualquer um dos métodos .then, o método .catch será acionado e o erro será exibido no console.
